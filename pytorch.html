PyTorch

Generative adversarial networks

  • Repository:

    • Context

    There is something of a machine learning craze in tech at the moment. Many people at my college are doing machine learning postgraduate degrees. In this code, I tried a slice of machine learning by creating my first generative adversarial network (GAN) using the PyTorch machine learning library. I used a GAN to generate images of handwritten digits. This involved using the torchvision package, installed via conda. The GAN uses two neural networks: A network to generate images, and a network to discriminate between real and fake images. In this way the generator network is trained to fool the discriminator network. After being fed training data, this GAN can generate images of a series of digits. My friend learnt how to encrypt a password into a GIF, so I see machine learning as something neat to learn about. I am not sure if I will pursue machine learning much further for now.

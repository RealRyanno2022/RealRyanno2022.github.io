Wordle

Web Game Development

  • Repository:

    • Context

    This project is a long time in the making. I first started trying to make a Wordle project last summer after learning React for the first time through a course, but this was ultimately unsuccessful. I decided to reboot the project early this year to finish what I started. The application has all the hallmarks of Wordle - The flipping letters, word dictionary, color-changing letters and keyboard which is responsive to both keystrokes and clicks. I do believe in the software adage that the last 10% of a project ends up taking 90% of the overall project time, however. This project is just a small bit rough around the edges. It would be much more practical to learn new frameworks and the like I found myself spending hours trying to fix simple logic mistakes in the code. JavaScript is not the most ideal programming language for creating complicated game logic, but it sure is great at making responsive web pages. I am proud of the final product, and I hope you enjoy playing it as much as I enjoyed making it.

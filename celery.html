Django Celery E-Mail App

Asynchronous E-Mail Sending

  • Repository:

    • Context

    I completed a project utilising the Python web framework Django and Celery, an asynchronous task queuer. This allowed me to send e-mails automatically. I personally found Django difficult to work with at first. You must use a venv terminal to execute python commands. However, the benefit of being able to use Python in a web framework cannot be underestimated. The app also has a basic "How are we doing?" e-mail sending interface which is a nice touch. I also used the Django REST framework to create a REST API for the app. This was a great learning experience and I would like to work with Django in the future.

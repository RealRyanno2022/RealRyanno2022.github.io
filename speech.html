Speech Recognition

Using Python

Context

In this project, I used the SpeechRecognition library in Python to capture microphone input and translate it into words which can be read in the Python terminal. used the Google Speech Recognition API to translate the audio into text. To flesh out the project, I made a game which allows you to guess the word said from a pre-downloaded audio prompt and a list of possible words. The game is played by saying the word you think is correct into the microphone. The game will then tell you if you are correct or not in the terminal. This project represented my foray into "Internet of Things" technology, which is an exciting and still infantile space.

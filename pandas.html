Pandas Gradebook

Dataframes and CSV Manipulation

Context

Pandas is a python data analysis library. I first encountered it in my Software Applications I module where I studied Python and was tasked with presenting data in assignments, exams and exercises. As a data visualisation method I would prefer R for complex data or most likely Excel, but pandas does the job. In this project, I take a CSV file containing student grades across a number of exams, quizzes and graded homework sessions, while also analysing average grades for each. I then take this data and put it into pandas dataframe structures. The final grades are saved as separate CSV files, and I achieved a grade distribution. I made use of pandas merge(), join() and concat() functions. Overall I was very happy about how this project turned out and I probably would use Pandas again.

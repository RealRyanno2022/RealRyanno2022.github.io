URL Shortener

Made using FastAPI and Python

  • Repository:

    • Context

    This project uses a SQLite database, FastAPI configuration and a Python backend to shorten URLs. The application ultimately returns some JSON code which has an "admin_url" and a "short_url". The admin_url is used to access the admin page for the shortened URL, where the user can see the number of times the URL has been accessed, the original URL and the shortened URL. The short_url is the URL that the user will use to access the original URL. The application also handles CRUD operations - being able to perform them on the URLs, as well as count the number of times the shortened URLs were used.

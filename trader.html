Trading Bot

Created using QuantConnect

Context

I first became interested in this project after hearing that a classmate's friend had scored a lucrative internship with SIG to develop Python trading algorithms over the summer. I decided to use the algorithmic trading platform QuantConnect and a demo account to test an algorithm which I had built using a Youtube tutorial. The bot tracks the SPX index and uses the last six years of data. It begins with a stop loss of 5% and starts the user with 100,000 dollars in virtual cash.

I had some initial problems debugging this project, but it went OK.
